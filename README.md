# Car-Dealership-YTD-Performance
## 📌 Project Overview

This project presents a Car Dealership YTD Performance Dashboard (2025) built using Power BI. It provides a comprehensive view of sales performance, revenue trends, and operational insights across dealerships, regions, and product categories.

The dashboard is designed with two interactive views (Overview & Detail Page) to help stakeholders quickly monitor KPIs and drill down into granular data for deeper analysis.

❗ Problem Statement 

Car dealership businesses handle large volumes of sales and operational data but lack a centralized system to:

- Track Year-to-Date (YTD) performance
- Compare performance with previous year
- Identify top-performing regions, dealers, and car categories
- Monitor sales trends and fluctuations

## 📌 Project Highlights - 
This project involved the following tasks:
- 	connecting and pre-processing the raw data using Power Query
- 	building a relational data model and report in Power BI Desktop 
- 	creating measures using DAX
- 	File Format – .xlsx for data source and .png for dashboard previews.

 
⚙️ Key Features
- 📊 KPI tracking (Revenue, Cars Sold, Avg Sale Value)
- 📅 YTD & MTD performance comparison
- 📈 Weekly revenue trend analysis
- 🌍 Regional performance breakdown
- 🚘 Sales segmentation by Body Style & Color
- 🏢 Company-wise performance analysis
- 🔎 Drill-through detail page for transaction-level insights
- 🎛️ Interactive filters (Body Style, Dealer Name, Transmission, Engine)
- 📊 Executive Summary (Overview Page)

## 📈 Key Insights
- 🚀 Revenue and sales volume are growing strongly YoY, indicating business expansion
- 📉 Average sale value slightly declined, suggesting a shift toward budget vehicles or discounts
- 🚘 SUVs are the top revenue-generating segment
- 🎨 Pale White is the most preferred car color among customers
- 🌍 Certain regions (Austin, Janesville) significantly outperform others
- 🏢 A few companies (Chevrolet, Dodge, Ford) dominate total sales contribution
- 📊 Weekly trends show spikes and volatility, indicating seasonal or promotional effects
